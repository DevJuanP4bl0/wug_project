# Sistema de criação de Wugs

Projeto em C que implementa a criação e gerenciamento de criaturas chamadas "Wugs".
O sistema permite gerar indivíduos com diferentes combinações genéticas, classifica-los por tipos, realizar o rankeamento e gerenciar uma população completa. Também, conta com funcionalidades de ordenação, geração de relatórios estatisticos e a conversão da genética para formato de string para exibição ao usuário.
